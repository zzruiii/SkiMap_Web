# 法国滑雪场比较网站

这是一个交互式的滑雪场比较网站，专为滑雪爱好者设计，帮助他们找到最佳的滑雪目的地。

## 🎿 产品目标
通过视觉化和交互式的方式比较滑雪场数据，为滑雪爱好者提供决策支持。

## 🧊 目标用户
寻求最佳滑雪目的地的休闲和专业滑雪者。

## 📄 页面结构
1. **首页** - 风格化雪山场景，伴随飘落雪花动画和滚动提示
2. **地图页** - 交互式地图，显示法国滑雪场（从南到北排列），悬停显示玻璃态信息卡
3. **图表页** - 柱状图比较各滑雪场的"蓝鸟日"，可拖拽时间选择器进行月/周过滤

## 🎨 设计特色
- 基于Figma设计的玻璃态效果
- 清晰的排版和流畅的动画过渡
- 主要高亮色：#4579CC

## 💻 技术栈
- **React 18** + **TypeScript** - 现代化前端框架
- **Tailwind CSS** - 快速样式开发
- **Framer Motion** - 高质量动画库
- **Vite** - 快速构建工具

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📁 项目结构
```
src/
├── components/          # React组件
│   ├── HomePage.tsx     # 首页组件
│   ├── Snowflake.tsx    # 雪花动画组件
│   └── ScrollIndicator.tsx # 滚动提示组件
├── assets/              # 静态资源
│   └── *.svg           # 从Figma导出的山脉SVG
├── App.tsx             # 主应用组件
├── main.tsx            # 应用入口
└── index.css           # 全局样式
```
