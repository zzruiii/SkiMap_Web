# 第三页功能 - 蓝鸟天数统计图表

## 功能概述

第三页已成功开发完成，并已集成真实蓝鸟天数数据，具备以下功能特性：

### 🎯 核心功能
1. **滑雪场图标排列** - 小山图标按从南到北（地理位置）从左到右依次排列在页面底部
2. **蓝鸟天数统计图表** - 显示各滑雪场在不同时间段的真实晴天数据
3. **交互式时间选择器** - 用户可选择不同月份和周数查看数据
4. **视觉数据点** - 白色圆点标记每个数据点位置
5. **动画效果** - 平滑的页面切换和图表动画

### 📊 图表特性
- **横坐标**: 各个滑雪场（按地理位置从南到北排序）
- **纵坐标**: Days/Week（每周天数，0-7天）
- **数据精度**: 保留小数点后一位
- **颜色编码**: 每个柱状图使用对应滑雪场的专属颜色
- **网格线**: 辅助读取数据的参考线

### 📅 数据时间范围
- **月份**: November, December, January, February, March, April
- **周数**: Week 1, Week 2, Week 3, Week 4
- **数据来源**: `bluebird_day.csv` 真实蓝鸟天数统计

### 🎮 用户交互
- **月份选择**: November, December, January, February, March, April
- **周选择**: Week 1, Week 2, Week 3, Week 4
- **实时更新**: 选择不同时间段时图表数据实时更新
- **动画反馈**: 选择按钮有视觉反馈效果

### 🚀 页面导航
- **滚动操作**: 
  - 从首页向下滚动 → 地图页
  - 从地图页向下滚动 → 图表页（第三页）
  - 从图表页向上滚动 → 地图页
  - 从地图页向上滚动 → 首页

## 📝 使用说明

### 如何访问第三页
1. 启动开发服务器：`npm run dev`
2. 打开浏览器访问 `http://localhost:5174`
3. 从首页开始，向下滚动两次到达第三页

### 如何使用图表
1. **选择月份**: 点击顶部月份按钮（Nov, Dec, Jan, Feb, Mar, Apr）
2. **选择周数**: 点击周选择按钮（Week 1-4）
3. **查看数据**: 观察柱状图高度和白色数据点位置
4. **读取数值**: 每个数据点上方显示具体数值

### 滑雪场识别
- 底部小山图标显示所有滑雪场
- 图标颜色与柱状图颜色对应
- 图标下方标注滑雪场名称
